FIR digital bandpass filter coefficients computation function.

Parameters
[Fs]sampling frequency , Hz
[Fp]pass frequency , Hz
[BW]bandpass width , Hz
[N]filter order
[scale]scaling factor ( scale = 0 - not-scaled coefficients , scale = 1 - scaled coefficients )
[wind]input window function object
Returns
The function returns digital highpass FIR filter coefficients buffer. Even order FIR bandpass filter coefficients are computed as follows:

\[ F_{p1} = \frac{ F_{p1} }{ Fs } \newline F_{p2} = \frac{ F_{p1}+BW }{ Fs } \newline n \in \left[ 0 ; \frac{ N }{ 2 } \right] \newline k = \left | n - \frac{ N + 1 }{ 2 } \right | \newline a_k = \begin{cases} 2 * ( F_{p1} - F_{p2} ) * w_i \quad , \quad n = 0 \\ 2 * \left[ F_{p2} * \frac{ \sin{ ( 2 * \pi * F_{p2} * n ) } }{ 2 * \pi * F_{p2} * n } - F_{p1} * \frac{ \sin{ ( 2 * \pi * F_{p1} * n ) } }{ 2 * \pi * F_{p1} * n } \right] * w_i \quad , \quad n > 0 \\ a_{N-k} = a_k \quad , \quad n > 0 \end{cases} \]

Odd order FIR bandpass filter coefficients are computed as follows:

\[ F_{p1} = \frac{ F_{p1} }{ Fs } \newline F_{p2} = \frac{ F_{p1}+BW }{ Fs } \newline n \in \left[ 0 ; \frac{ N }{ 2 } \right] \newline k = \left | n - \frac{ N + 1 }{ 2 } \right | \newline r = ceil\left(n - \frac{N}{2} \right) - \frac{N}{2} \newline a_k = 2 * \left[ F_{p2} * \frac{ \sin{ ( 2 * \pi * F_{p2} * ( n + r ) ) } }{ 2 * \pi * F_{p2} * ( n + r ) } - F_{p1} * \frac{ \sin{ ( 2 * \pi * F_{p1} * ( n + r ) ) } }{ 2 * \pi * F_{p1} * ( n + r ) } \right] * w_i \newline a_{N-k} = a_k \quad , \quad n > 0 \]

If the flag scale = 1, then coefficients are scaled as follows:

\[ n \in \left[ 0 ; N \right] \newline a_n = \frac{ a_n }{ \left| W(j*\omega) \right| } \]

[scale]scaling factor ( scale = 0 - not-scaled coefficients , scale = 1 - scaled coefficients )

The function returns digital lowpass FIR filter coefficients buffer. Even order FIR highpass filter coefficients are computed as follows:

\[ Fp = \frac{ Fp }{ Fs } n \in \left[ 0 ; \frac{ N }{ 2 } \right] \newline k = \left | n - \frac{ N + 1 }{ 2 } \right | \newline a_k = \begin{cases} ( 1 - 2 * F_p ) * w_i \quad , \quad n = 0 \\ ( 1 - 2 * F_p ) * \frac{ \sin{ ( 2 * \pi * n * F_p ) } }{ 2 * \pi * n * F_p } * w_i \quad , \quad n > 0 \end{cases} a_{N-k} = a_k \quad , \quad n > 0 \]

Odd order FIR highpass filter coefficients are computed as follows:

\[ Fp = \frac{ Fp }{ Fs } n \in \left[ 0 ; \frac{ N }{ 2 } \right] \newline k = \left | n - \frac{ N + 1 }{ 2 } \right | \newline r = ceil \left( n - \frac{N}{2} \right) - \frac{N}{2} \newline a_k = -1^{n} * 2 * \left[ 0.5 - F_p \right] * \frac{ \sin{ \left[ ( n + r ) * 2 * \pi * ( 0.5 - F_p ) \right] } }{ \left[ ( n + r ) * 2 * \pi * ( 0.5 - F_p ) \right] } \]

If the flag scale = 1, then coefficients are scaled as follows:

\[ n \in \left[ 0 ; N \right] \newline a_n = \frac{ a_n }{ \left| W(j*\omega) \right| } \]
